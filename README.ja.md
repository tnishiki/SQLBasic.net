# SQLBasic.net

> [英語](README.md) | **日本語** | [簡体字中国語](README.zh-CN.md) | [スペイン語](README.es.md)

**SQLBasic.net** は **SQLite** 向けの直感的な SQL エディタです。
シンタックスハイライト・スマート自動補完・SQL 整形を備え、実行結果を同一画面内で即座に確認できます。
クラウド接続不要で完全オフラインで動作するため、セットアップが最小限で済み、インストール直後からすぐに学習に集中できます。

## 利用者の想定

このアプリケーションは **これから SQL を学ぼうとする初心者** を主な対象としています。
データベースのインストールや設定は一切不要です。

起動するだけで学習用の SQLite ファイルが自動生成されるため、すぐにテーブル作成やデータ操作を体験できます。
シンタックスハイライトとスマート自動補完により構文ミスを早期に発見でき、操作に不慣れな段階でも安心して使えます。
完全オフラインで動作するため、ネットワーク環境に左右されず自分のペースで学習を進められます。

## 動作環境

- Windows 10 / 11
- .NET 8.0 (Windows デスクトップ)

## 機能一覧

### エディタ

| 機能 | ショートカット | 説明 |
|---|---|---|
| シンタックスハイライト | — | SQL キーワード・文字列・数値・コメントをそれぞれ色分け表示 |
| SQL 整形 | `Alt+F` | ドキュメント全体を整形 |
| 行コメント | `Ctrl+K` | 現在行または選択行の行頭に `-- ` を追加 |
| 行コメント解除 | `Shift+Ctrl+K` | 現在行または選択行の行頭から `-- ` を削除 |
| カーソル位置クエリ実行 | `Ctrl+Enter` | セミコロンを区切りとして、カーソル位置にあるクエリのみを実行 |
| CSV エクスポート | `Ctrl+L` | クエリ結果を CSV ファイルに書き出す |

### スマート自動補完（`Ctrl+Space`）

自動補完はコンテキストを判断し、カーソル位置に応じて適切な候補を提示します。

| カーソル位置 | 補完候補 |
|---|---|
| `FROM` / `JOIN` / `UPDATE` / `INSERT INTO` の直後 | テーブル名 |
| `DROP TABLE` / `ALTER TABLE` / `CREATE TABLE` の直後 | テーブル名（`IF EXISTS` / `IF NOT EXISTS` にも対応） |
| `SELECT` の列リスト中 | クエリで参照しているテーブルのカラム名 |
| `テーブル名.` の直後 | そのテーブルのカラム名 |
| `SELECT` 列名の直後（`FROM` の前） | `FROM` |
| 文の先頭（新規ステートメント） | `SELECT`、`INSERT INTO`、`UPDATE`、`DELETE FROM`、`CREATE TABLE` など |
| `FROM` テーブル名の直後 | `WHERE`、`INNER JOIN`、`LEFT JOIN`、`GROUP BY`、`ORDER BY`、`LIMIT` など |
| `WHERE` / `AND` / `OR` の直後 | `AND`、`OR`、`NOT`、`EXISTS`、`BETWEEN`、`IN`、`LIKE`、`IS`、`NULL` |
| `GROUP` / `ORDER` の直後 | `BY` |
| `ORDER BY` カラム名の直後 | `ASC`、`DESC`、`NULLS FIRST`、`NULLS LAST` |
| `INNER` / `LEFT` / `RIGHT` / `FULL` の直後 | `JOIN`、`OUTER JOIN` |
| `LIMIT` の直後 | `OFFSET` |
| `CREATE` の直後 | `TABLE`、`INDEX`、`VIEW`、`TRIGGER`、`TEMP TABLE` |
| `DROP` の直後 | `TABLE`、`INDEX`、`VIEW`、`TRIGGER` |

### 実行結果メッセージ

クエリ実行後、ウィンドウ下部のステータスバーに結果メッセージを表示します。

| 操作 | メッセージ |
|---|---|
| `SELECT` | ヒットした件数 |
| `INSERT` | 挿入した件数 |
| `UPDATE` | 更新した件数 |
| `DELETE` | 削除した件数 |
| `CREATE TABLE` / `CREATE INDEX` | 作成したオブジェクトの種別と名前 |
| `DROP TABLE` / `DROP INDEX` | 削除したオブジェクトの種別と名前 |

### データベースパネル（右サイドバー）

- **テーブル一覧** — 接続中の SQLite データベースに存在するテーブルを一覧表示
- **カラム情報** — テーブルを選択すると、カラム名・データ型・NULL 許可の情報を表示
- **他の DB に接続** — 任意のタイミングで別の SQLite ファイルに切り替え可能

### その他

- **履歴管理** — 過去に実行したクエリを参照・再実行できる
- **クエリテンプレート** — よく使う SQL スニペットを登録・再利用できる
- **複数ウィンドウ** — 複数のエディタウィンドウを開いてクエリを比較できる
- **DB の自動生成** — 初回起動時に SQLite データベースファイルを自動作成

## キーボードショートカット

| ショートカット | 操作 |
|---|---|
| `Ctrl+Enter` | カーソル位置のクエリを実行 |
| `Ctrl+Space` | 自動補完を開く |
| `Ctrl+K` | 行コメントを追加（`-- `） |
| `Shift+Ctrl+K` | 行コメントを解除 |
| `Alt+F` | SQL ドキュメントを整形 |
| `Ctrl+L` | 結果を CSV にエクスポート |

## プログラムの起動

起動すると、指定のパスに SQLite データベースファイルが自動的に作成されます。
その後、組み込みエディタで SQL クエリを記述・実行できます。

別の SQLite ファイルに接続するには、右サイドバーの **「他のDBに接続する」** ボタンをクリックしてください。
